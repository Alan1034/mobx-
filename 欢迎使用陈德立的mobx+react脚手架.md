# 欢迎使用陈德立的mobx+react脚手架

使用前请：

1.cnpm install 安装依赖

2.npm-check检查更新

npm-check

3.npm-upgrade更新

npm update --save

4.npm start 开始使用

5.注意，请备份后修改脚手架（对作者陈德立）
